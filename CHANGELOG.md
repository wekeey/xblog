# 更新日志

所有项目的重要变更都会记录在这个文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-08-21

### 新增

- 基于 Next.js 15 的博客系统
- MDX 内容管理支持
- 全文搜索功能（支持中文分词）
- 响应式设计
- 明暗主题切换
- SEO 优化
- Vercel Analytics 集成

### 技术栈

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Fumadocs MDX
- shadcn/ui
- Jotai
- Shiki
